# Project-Browser-Extension-to-Block-Trackers
A privacy-focused browser extension that blocks known tracking scripts and analytics to protect your online privacy.

## Features
- Blocks known tracking domains (Google Analytics, Facebook Pixel, etc.)
- Real-time tracker counter
- Lightweight and fast
- Works on Chrome, Edge, and other Chromium-based browsers
- Privacy-first design with no data collection

## Installation

### For Chrome/Edge:
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the project folder
6. The extension is now installed!

### For Firefox:
1. Go to `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on"
3. Select the `manifest.json` file
